# Rubio

The Rdio API gem.
I wanted a good way to access the Rdio API. I only found gem written in Ruby that seemed to do it. But I wanted to make my own.

## Installation

Add this line to your application's Gemfile:

    gem 'rubio'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rubio
